📝 To-Do List App
A clean, responsive To-Do List web application built with vanilla JavaScript and Bootstrap 5. Manage your daily tasks easily with a simple and beautiful UI.

🌐 Live Demo

Open to-do-list.html directly in any browser — no server needed!


✨ Features

✅ Add new tasks instantly (click button or press Enter)
✏️ Edit existing tasks with a single click
🗑️ Delete tasks with smooth fade-out animation
☑️ Mark tasks as completed (strikethrough effect)
🔢 Live pending task counter
📌 Empty state message when no tasks exist
📱 Fully responsive — works on mobile, tablet & desktop
🎨 Custom edit & delete icons
⚡ No frameworks, no installation — pure HTML, CSS & JS
